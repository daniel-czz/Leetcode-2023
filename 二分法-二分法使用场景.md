
# 二分法使用场景
1. 经典查找问题（数组有序）       
	•	描述：在一个有序数组中查找某个目标值。    
	•	例子：    
	•	查找元素是否存在（Leetcode 704. Binary Search）     
	•	查找左/右边界（Leetcode 34. 在排序数组中查找元素的第一个和最后一个位置）

2. 单调性问题（最小最大值）         
	•	描述：在一个范围 [left, right] 内查找 满足某种条件的最小/最大值。           
	•	例子：              
	•	找某个值的最小上界/最大下界     
	•	最小化最大值（如 Leetcode 410. 分割数组的最大值）       
	•	最大化最小值（如 Leetcode 875. 爱吃香蕉的珂珂）
